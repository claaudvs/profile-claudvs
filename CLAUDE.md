# Project Context

This project is a personal developer portfolio built with Astro.

Sections included:

- Home
- About Me
- Skills
- Projects
- Blog
- Contact

The design should be minimal, modern and developer-focused.

# Stack

- Astro
- TypeScript
- TailwindCSS
- Markdown for blog posts

# Goals

Create a fast, clean and elegant developer portfolio.

Important principles:

- minimal UI
- readable typography
- fast loading
- reusable components

# Content

Projects will describe technical challenges and solutions without revealing confidential company information.

# Coding rules

- Use reusable Astro components
- Keep components small
- Use semantic HTML
- Prefer simple CSS with Tailwind
